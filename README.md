# Video Snapshot

Python + OpenCV を使って、動画を再生しながらミリ秒精度のタイムスタンプ入りでスナップショットを保存できるツールです。

## 特徴

* 起動直後は一時停止状態
* スペースキーで再生 / 一時停止
* `s` キーでスナップショット保存（動画名＋経過時間＋フレーム番号）
* `a` / `d` キーで一時停止中にフレーム送り / 戻し
* `r` キーで画面幅を維持したままアスペクト比を補正
* `R` キーでウィンドウサイズを元のフレームサイズにリセット
* `q` またはウィンドウの × ボタンで終了

## Usage

### コマンドライン引数あり

```bash
uv run python main.py path\to\video.mp4
```

### コマンドライン引数なし

```bash
uv run python main.py
```

* ファイル選択ダイアログが開きます。

## スナップショットの保存場所

* 元動画ファイルと同じフォルダ内に `snapshots` フォルダを自動作成
* 保存形式: PNG
* ファイル名例: `動画名_0001m12s345ms_000123.png`

## 操作方法

| キー    | 動作                      |
| ----- | ----------------------- |
| space | 再生 / 一時停止               |
| s     | スナップショット保存              |
| a     | 一時停止中に 1 フレーム戻す         |
| d     | 一時停止中に 1 フレーム進める        |
| r     | 現在の幅を維持してアスペクト比を補正      |
| R     | ウィンドウサイズを元のフレームサイズにリセット |
| q / × | 終了                      |

## 注意点

* Windows の場合、OpenCV がウィンドウサイズを自動調整する際、ディスプレイのスケーリング設定の影響でアスペクト比が崩れることがあります。
* `uv` コマンドで起動する場合は、Python 仮想環境やパスが正しく設定されていることを確認してください.
